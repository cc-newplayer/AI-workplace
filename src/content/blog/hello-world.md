Welcome to my blog! This site is powered by **DeerFlow** and **Astro**, automatically publishing AI-generated articles to GitHub Pages.

## What is this?

This blog is automatically updated by an AI agent that researches topics, writes SEO-optimized articles, and publishes them here via the GitHub API.

## How it works

1. DeerFlow agent researches a keyword
2. Writes a structured article with SEO metadata
3. Commits the Markdown file to this repository
4. GitHub Actions rebuilds and deploys the site
